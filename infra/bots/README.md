Skia Infrastructure
===================

This directory contains infrastructure elements:

  * assets - Artifacts used by automated tests are versioned here, along with
      scripts for recreating/uploading/downloading them.
  * ct - Helpers for running Skia tasks in Cluster Telemetry.
  * gen_tasks.go - Run this to regenerate tasks.json after changing any of the
      JSON files in this directory or updating any assets.
  * recipe_modules - Shared modules which are used by recipes.
  * recipes - Scripts which run inside of Swarming tasks.
  * tasks.json - This file determines what tasks are run on Skia's
      infrastructure and how. It is generated by gen_tasks.go.
  * tools - Assorted other infrastructure-related tools, eg. isolate and CIPD
      binaries.
  * isolate files - Used when triggering Swarming tasks to determine which files
      to send to the bot.
