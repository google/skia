Canvas Path Editor

A typographical demonstration of the experimental Canvas 2D TextClusters and advanced TextMetrics APIs. Type, click, drag, and navigate text laid out dynamically along curved geometries.

⚠️ Experimental APIs Not Detected: Your browser does not currently expose TextMetrics.prototype.getTextClusters or CanvasRenderingContext2D.prototype.fillTextCluster. The demo is running in linear fallback mode. To experience true grapheme cluster orientation, please launch Chrome with --enable-experimental-web-platform-features.
Path & Typography
💡 Editing Tips Click anywhere on the curved text to place the caret. Drag to select a character range. Use keyboard arrows (←/→) to step between minimal grapheme clusters.
